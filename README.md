## ✨ 功能特性

### 📚 作业管理
- **富文本编辑**: 支持格式化文本的作业内容编辑
- **截止日期管理**: 清晰的日期显示和过期提醒
- **状态跟踪**: 自动标识过期、即将到期的作业
- **快速操作**: 便捷的编辑、删除功能

### 🏷️ 分类系统
- **学科管理**: 自定义学科分类，支持增删改查
- **标签系统**: 灵活的标签管理，支持多标签分类
- **智能分组**: 按学科自动分组显示作业

### 🎨 界面定制
- **主题切换**: 支持明暗主题切换
- **界面缩放**: 50%-200% 自由缩放界面
- **列数调整**: 1-5 列灵活布局
- **透明度控制**: 可调节背景透明度
